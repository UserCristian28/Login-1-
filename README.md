Minimalist Login Page

Una página de inicio de sesión moderna y responsiva con un diseño estético y minimalista.

Características
* Diseño Limpio: Interfaz minimalista inspirada en interiores modernos.
* Campos Personalizados: Inputs con bordes redondeados, efectos de transparencia y estados de enfoque.
* Tipografía Elegante: Integración de Google Fonts (Montserrat).

Tecnologías Utilizadas
* HTML5: Estructura semántica.
* CSS: Estilos personalizados, posicionamiento absoluto/relativo y variables de fondo.
* Google Fonts: Fuente "Montserrat" para un acabado profesional.

Instalación y Uso
1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/UserCristian28/Login-1-.git]
    ```
    
2.  Abre el archivo:
    Simplemente abre `index.html` en tu navegador preferido.

Ajustes Personalizados

Para cambiar el tamaño o la posición de la imagen de fondo, dirígete al archivo CSS y modifica la sección `.wrapper`:

```css
.wrapper {
    background-size: cover; /* Cambia a un valor en px para ajustar el tamaño */
    background-position: center;
}
